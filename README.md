# Wordle
- Este juego costo 1 millon de dolares

## Descompsicion y Pasos

- se debe de ejecutar todo en una funcion
- cada fragmento del codigo debe de ejecutarse en una funcion aparte
- debe de preguntarle al usuario un string
- se usara el string recibido como un array para saber la posicion de la letra
- una condicional que aplique la regla: el input debe de tener un limite de caracteres para que se valido, si no es valido dar respuesta negativa y no usuar ese string
- a ese string se le cuentan los caracteres
- si tiene una de las letras debe de indicarte que letras tiene validas, por ende el input debe de compararse con la variable de respuesta correcta
- tiene que tener un contador de intentos que se almacenen en una variable, cuando la variable llegue a un maximo de intentos, se debe de parar la ejecucion de la funcion (condicional) y mandar un print indicando 'Game Over! /n/nTry again!'
